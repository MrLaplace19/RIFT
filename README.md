# RIFT Chat: Простой консольный чат на Python

Привет! Это RIFT Chat – мой небольшой проект консольного чата, написанный на Python с использованием вебсокетов. Идея была создать что-то простое, но функциональное, чтобы можно было пообщаться с друзьями прямо из терминала.

## О проекте

RIFT Chat позволяет пользователям регистрироваться, входить в систему и обмениваться сообщениями. Есть как общий чат для всех, так и возможность отправлять личные сообщения. Вы всегда можете увидеть, кто сейчас онлайн.

## Возможности

*   **Регистрация и Аутентификация**: Создайте аккаунт и входите в чат под своим логином и паролем.
*   **Общий чат**: Отправляйте сообщения, которые увидят все участники.
*   **Личные сообщения (ЛС)**: Общайтесь приватно с конкретным пользователем. Просто используйте команду `/pm`.
*   **Список онлайн-пользователей**: Узнайте, кто из ваших друзей сейчас в сети с помощью команды `/users`.

## Установка и запуск

Чтобы запустить этот чат, вам понадобится Python 3.9+ и `uv` (или `pip` с `venv`).

### 1. Клонируем репозиторий

```bash
git clone https://github.com/MrLaplace19/RIFT.git # Замените на актуальный URL
cd RIFT
```

### 2. Настройка окружения и установка зависимостей

Я предпочитаю `uv` для управления зависимостями, но `pip` тоже подойдет.

```bash
uv sync # Или pip install -r server/requirements.txt
```

### 3. Запуск сервера


Для этого нужен установленный Docker и Docker Compose.
```bash
docker-compose up --build
```
Сервер будет доступен на `ws://localhost:8765`.

### 4. Запуск клиента

Клиент — это то, через что вы будете общаться. Откройте **новое окно терминала** после запуска сервера.

```bash
python client/clients.py --host localhost --port 8765
```

При первом запуске вам будет предложено зарегистрироваться (`r`) или войти (`l`).

## Использование команд в чате

*   Отправка **общего сообщения**: Просто введите текст и нажмите Enter.
*   **Личное сообщение**: `/pm <имя_получателя> <ваше_сообщение>`
    *Пример: `/pm Alice Привет, как дела?`*
*   **Посмотреть список онлайн-пользователей**: `/users`
*   **Выйти из чата**: `exit` или `Ctrl+C`

## Технологии

*   **Python 3.9+**
*   **websockets**: Для асинхронной связи между клиентом и сервером.
*   **asyncio**: Для асинхронного выполнения задач.
*   **argparse**: Для обработки аргументов командной строки в клиенте.
*   **getpass**: Для безопасного ввода пароля.
